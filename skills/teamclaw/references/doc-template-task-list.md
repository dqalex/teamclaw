## 语法说明

| 标记 | 状态 | 优先级 |
|------|------|--------|
| `[ ]` | todo | medium |
| `[!]` | todo | high |
| `[-]` | todo | low |
| `[~]` | in_progress | - |
| `[?]` | reviewing | - |
| `[x]` | completed | - |

**其他语法**：
- `@成员名` — 分配任务
- `[[文档名]]` — 关联文档（可多个）
- `[[doc:xxx]]` — 关联文档（ID 匹配）
- `#task_xxx` — 引用已有任务
- `[进度%]` — 进度百分比
- `> 描述` — 任务描述（4 空格缩进）
- `> 截止日期: YYYY-MM-DD` — 截止日期
- 4 空格缩进 `- [ ]` — 子任务清单（标准嵌套列表语法）

**子任务缩进规范**：
子任务必须使用 **4 空格缩进**（标准 Markdown 嵌套列表），确保在各种 Markdown 编辑器中正确渲染为嵌套结构：
```markdown
- [ ] 主任务
    - [ ] 子任务1
    - [x] 子任务2
```
