👥

Members

人类/AI 成员管理，AI 成员含 Gateway Agent 绑定、经验数据、API Token 管理

📝

Wiki

Markdown 文档编辑器，8 种文档类型（guide/report/note/decision/scheduled_task/task_list/reference/other）

📦

Deliveries

文档交付审核流程（pending → approved/rejected/revision_needed），支持内部文档和外部链接

🤖

Agents

OpenClaw Agent 管理，创建/删除/文件管理，需 Gateway 连接

💬

Sessions

Gateway 会话列表，支持 thinking/verbose/reasoning 级别设置

⏰

Schedule

Gateway Cron Job 管理（创建/编辑/启停/手动执行），执行历史查看

🧩

Skills

Gateway Skills 查看/启停/安装

⚙️

Settings

语言切换、主题切换、Gateway 配置面板、Workspace 管理

文件同步特性：
• 版本控制：每次变更保存完整/差异版本到 openclaw_versions 表
• 冲突检测：本地和远端同时修改时进入 openclaw_conflicts，支持手动解决
• 增量同步：基于 .teamclaw-index 的文件 hash 对比，只同步变更文件
• 文件监听：watcher.ts 实时监控 workspace 文件变更（排除 node_modules/.git/tasks 等）
